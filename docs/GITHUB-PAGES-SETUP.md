# GitHub Pages Setup

Host your privacy policy, terms of service, and delete account pages for free using GitHub Pages.

## Steps

1. Push all changes to GitHub (the `docs/` folder must be on the `master` branch)

2. Go to your GitHub repository: https://github.com/ateeqpirzada01-cyber/life-in-uk-app

3. Click **Settings** (top menu bar)

4. Click **Pages** (left sidebar, under "Code and automation")

5. Under **Source**, select:
   - **Deploy from a branch**
   - Branch: **master**
   - Folder: **/docs**

6. Click **Save**

7. Wait 1-2 minutes for deployment

## Your URLs

Once enabled, your pages will be available at:

| Page | URL |
|------|-----|
| Landing | https://ateeqpirzada01-cyber.github.io/life-in-uk-app/ |
| Privacy Policy | https://ateeqpirzada01-cyber.github.io/life-in-uk-app/privacy-policy.html |
| Terms of Service | https://ateeqpirzada01-cyber.github.io/life-in-uk-app/terms-of-service.html |
| Delete Account | https://ateeqpirzada01-cyber.github.io/life-in-uk-app/delete-account.html |

## Where to Use These URLs

- **Google Play Console** → Store listing → Privacy policy URL
- **Google Play Console** → App content → Data deletion policy URL
- **In the app** → Profile/Settings screen → Privacy Policy link

## Verify

After enabling, visit the privacy policy URL in your browser to confirm it loads correctly.
